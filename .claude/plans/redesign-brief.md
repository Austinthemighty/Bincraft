# ItemCards Redesign Brief

## Resolved Decisions
- **Sidebar**: Collapsible with icons. Collapsed = icons-only (tablet/mobile). Expandable for labels. User preference persisted.
- **QR Codes**: Regenerated with configurable base URL from Settings. QR encodes scan URL. Item ID printed as plain text below QR as fallback.
- **Settings**: Add "App URL" field for QR generation base URL.
- **Theme**: Dark mode, blue primary accent
- **Personality**: Efficient, reliable, no-nonsense. Industrial precision.
- **All pages equally**: Full cohesive redesign.
